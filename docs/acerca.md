# Acerca de mí

Hola, soy Gonzalo Soto, estudiante de Ingeniería en Informática y actualmente en la recta final de la carrera. Me apasiona construir cosas que funcionen de verdad, aprender tecnologías nuevas y meter mano en proyectos que me desafíen. ## Habilidades

Mi enfoque está en el desarrollo backend usando Python, Java y C#, con una base sólida en programación orientada a objetos y buenas prácticas de diseño. He trabajado en procesos de ETL, integración de datos y automatización de tareas dentro de entornos productivos. También cuento con experiencia en el manejo y modelado de datos con SQL, utilizando PostgreSQL, SQL Server y MySQL, y complemento este trabajo con herramientas como Power BI para análisis y con Solr para búsqueda e indexación.
